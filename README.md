# 🏦 Consumer Complaint Intelligence Dashboard

An end-to-end Data Analytics project built using **Python, SQLite, SQL, and Power BI** to analyze consumer complaint data from financial institutions.

---

## 📌 Project Overview

This project analyzes over **14,000 consumer complaints** to identify complaint trends, product-wise complaint distribution, company performance, geographical insights, and complaint resolution metrics.

The project demonstrates the complete analytics workflow from data cleaning to interactive dashboard creation.

---

## 🎯 Business Objective

Financial institutions receive thousands of complaints every year.

The objective of this project is to transform raw complaint data into meaningful business insights that help answer questions such as:

- Which financial products receive the highest complaints?
- Which companies generate the most complaints?
- Which states report the highest complaint volumes?
- How do complaints vary over time?
- What is the average complaint resolution time?
